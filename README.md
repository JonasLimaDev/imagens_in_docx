# Divisor de PDF's

Adiciona imagens em um arquivo DOCX


## REQUISITOS
- Python 3
- python-docx
- GTK+3.0


## OPÇÕES
- Dimensões
  - são medidas em centímetros.
  - opções pré testadas.
  
- Legenda
  - quando ativa legenda será o nome da pasta final onde as fotos estão.
 

### Informações Adicionais de Funcionamento

- imagens serão adicionadas em formato de tabela.
- tenta ajustar a quantidade de imagens conforme o tamanho.
- sempre irá sobrescrever o arquivo final.
